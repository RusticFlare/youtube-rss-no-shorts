# 📺 YouTube RSS (No Shorts)

Paste a YouTube channel URL, get back an RSS feed that excludes Shorts.

## How it works

Finds the channel's hidden RSS link and swaps the channel ID prefix from `UC` to `UULF`, which is YouTube's long-video-only playlist format.

**Live demo:** https://rusticflare.github.io/youtube-rss-no-shorts/
